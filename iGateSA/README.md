# iGateSA - iGateServiceAgent

> Cara Build exe
1. Build exe file dapat menggunakan aplikasi Powershell
2. Wajib install SDK .Net 9
3. Jalankan command *dotnet build -c Release -o "./Exe/"*

> Contoh : PS F:\02 - DEVELOPMENT\05. iGate Sync Agent\GitTest\IGateSA_Windows_Service> dotnet build -c Release -o "./Exe/"
